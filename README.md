### 技术点
* 通过 jobqueue 实现缓存retrofit失败的请求(4.4 gms一直有问题，5.0 测试通过 待修改)
* 通过拦截器设置cookie (已实现)
* 搭建网络访问的基础工具类(已实现)
* 通过stetho网络调试(已完成)
